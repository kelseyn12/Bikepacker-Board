**Bikepacker-Board**
A full-stack CRUD application that allows the user to view
bikepacking routes in the midwest, update, edit, and delete.

**Technologies Used**
HTML 
CSS
JavaScript
Express
Mongoose
MongoDB
NodeJs
Heroku
GoogleFonts
